## How to run

1. Create a root project folder named Red-Ball
```
mkdir Red-Ball
```
2. Clone the repo
```
git clone https://github.com/PMafra/Red-Ball-Game.git
```
3. Install NPM packages for the repo
```
npm i
```
4. Create the dist folder:
```
npm run build
```
5. Run the app with
```
npm run dev
```
6. Open app in given localhost port
